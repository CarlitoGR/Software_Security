# Threat Model Starter

## Assets
- Patient personal information
- Medical history
- Admission records
- Login credentials
- Audit logs

## Example Threats
- Unauthorized viewing of patient records
- Weak authentication
- SQL injection or unsafe input handling
- Data leakage through logs or debug output
- Missing authorization checks
- Overly broad search results
- Insider misuse of clinician access
